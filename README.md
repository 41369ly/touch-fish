# touch-fish
idea插件 摸鱼神器 位置隐蔽难以发现可以放心摸鱼，内置老板键一键隐藏，自定义热键，自动切换章节，自动记录上次阅读位置，搜索关键词提示，在线阅读无需下载，附有书架功能
