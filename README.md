## 使用教程
1).到release中下载压缩包;

2).打开IDEA,将压缩包拖入IDEA,重启IDEA

3).进入IDEA的Setting页面，找到Other Settings\Fish Config选项,点击新增按钮，添加你想看的书

4).在idea中按下下一行热键

5).愉快的摸鱼吧！！！！
