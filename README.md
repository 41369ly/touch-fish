# touch-fish
#### idea插件 摸鱼神器 在线阅读无需下载 位置隐蔽难以发现

# 目前功能
1. 内置老板键一键隐藏
1. 自定义热键
1. 自动切换章节
1. 自动记录上次阅读位置
1. 搜索关键词提示
1. 书架功能
